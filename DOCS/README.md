## 项目简介

DeepSeek Free API 是一个将 DeepSeek Chat 接口转换为 OpenAI 兼容 API 的服务，支持：

- ✅ 高速流式输出
- ✅ 多轮对话
- ✅ 联网搜索
- ✅ R1 深度思考
- ✅ 静默深度思考
- ✅ 零配置部署
- ✅ 多路 Token 支持
- ✅ 完全兼容 OpenAI 接口

- **运行时**: Node.js 18+
- **框架**: Koa.js
- **语言**: TypeScript
- **构建工具**: tsup
- **测试框架**: Jest
